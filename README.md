## 🔍 What is paragon?

paragon is a dark theme designed to make your text editor and terminal easier on the eyes. It works with Neovim, VSCode, Alacritty, Kitty, WezTerm, and other popular tools. The theme uses clear colors that reduce eye strain and help you focus on your work. paragon also supports plugins to enhance your experience with these editors.

- Includes support for Neovim plugins and ports to popular terminal emulators

## ⚙️ Installing paragon theme on your applications

The steps to use paragon depend on the software. Follow the instructions for your editor or terminal below.

### Neovim

1. Place the downloaded color scheme file in your Neovim configuration folder. This is usually:  
   `C:\Users\<your-username>\AppData\Local\nvim\colors`

2. If you don’t have a `colors` folder, create it.

3. Open your Neovim config file (`init.vim` or `init.lua`) with a plain text editor.

4. Add this line to select the paragon theme:  
   For `init.vim`:  
   `colorscheme paragon`  
   For `init.lua`:  
   `vim.cmd('colorscheme paragon')`

5. Save the file and start Neovim. The dark theme should be active.

### Visual Studio Code

1. If you downloaded a `.vsix` file:  
   - Open VSCode.  
   - Go to the Extensions tab (square icon on the side).  
   - Click the three dots at the top-right of the Extensions pane.  
   - Select “Install from VSIX...” and choose the downloaded file.  
   - After installation, go to the Color Theme menu and select “paragon.”

2. If you see instructions or a link to the VSCode marketplace, follow those to install the theme.

### Alacritty

1. Open your Alacritty configuration file, usually named `alacritty.yml`. It is often located in your user directory (`C:\Users\<your-username>\`).

2. Add or replace the colors section with the content from the paragon Alacritty theme file you downloaded.

3. Save the config file and restart Alacritty.

### Kitty

1. Find your Kitty config file, often `kitty.conf` located in your user directory.

2. Copy the color settings from the paragon Kitty theme file you downloaded.

3. Paste them into your `kitty.conf` file or include the theme file with an `include` statement.

4. Save the changes and restart Kitty.

### WezTerm

1. Open your WezTerm config file, usually `wezterm.lua` located in your user directory.

2. Add or update the color scheme with the contents of the paragon WezTerm theme file.

3. Save the file and restart WezTerm.

## 🧩 Plugin Support with paragon (Neovim only)

paragon has built-in support for popular Neovim plugins that change how the editor looks and works. For example:

- **Telescope**: Makes file searching easier with paragon’s matching colors  
- **NvimTree**: Shows file trees with the paragon palette  
- **Lualine**: Status line uses paragon’s color scheme for visual consistency  

If you use these plugins, paragon automatically adjusts colors to fit them. No extra setup is needed once you install the main theme file.
