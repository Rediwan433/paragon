# FZF
Paragon theme for FZF fuzzy finder.

- **Bash/Zsh (`.sh`)**: add the Bash/Zsh to your `~/.bashrc`, `~/.zshrc` or custom file loaded on shell startup.
- **Fish (`.fish`)**: run the Fish snippet once.
- **Nushell (`.nu`)**: add the Nushell snippet to your `config.nu` file.
- **PowerShell (`.ps1`)**: add the PowerShell snippet to your `~/Documents/PowerShell/Mi
